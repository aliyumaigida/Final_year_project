Breast Cancer Detection

Using Machine Learning/Deep Learning to Support Early Diagnosis

About This Project

This breast cancer detection tool was developed using multiple machine learning models plus Deep learning CNN to classify tumors as either benign or malignant based on 30 diagnostic features. The models trained and evaluated include: Random Forest, XGBoost, Support Vector Classifier (SVC), Logistic Regression, Gaussian Naive Bayes, and Convolutional Neural Network (CNN) such as Conv1D, ResNet and Autoencoder.

Libraries used.
  • Pandas & NumPy – For feature extraction and manipulation
  • Python 3.12.7, Machine Learning (scikit-learn), Deep Learning(TensorFlow, Keras)
  • Django REST Framework (API) (Backend)
  • HTML, Bootstrap 5 (Frontend)
Purpose

The primary goal is to provide a simple and accessible tool for early detection of breast cancer, helping doctors and patients make informed decisions faster.
